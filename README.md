# 🚀 Robotics & Software Engineering Portfolio

An interactive, responsive web portfolio built from the ground up using **React** to showcase my engineering journey, research in autonomous systems, and web development capabilities. 

This platform serves as a bridge between high-fidelity robotics engineering—covering SLAM, simulation, and control theory—and modern, clean web architecture.

---

## 🛠️ Tech Stack & Architecture

### **Front-End Development**
* **Framework:** React.js (Component-driven, modular architecture)
* **Styling:** Clean, responsive design optimized for seamless cross-device viewing
* **Deployment:** [e.g., GitHub Pages / Vercel / Netlify]

### **Robotics Ecosystem Highlighted**
* **Frameworks & Simulation:** ROS2 Humble, Gazebo
* **Perception & SLAM:** ORB-SLAM3, OpenCV, Active Vision Systems
* **Languages:** C++, Python, JavaScript

---

## ⚙️ Core Engineering Philosophy

This portfolio is built to reflect the **"Chef" Philosophy**: prioritizing a deep, fundamental understanding of systems over blindly relying on pre-built, black-box solutions. Whether crafting a responsive UI component or troubleshooting a coordinate transform ($TF2$) in a custom robotic simulation, the focus is always on robust, ground-up logic and resourceful, frugal engineering.

---

## 📂 Key Projects Featured

* **Active Structured Light SLAM:** Research-focused implementation mapping textureless environments using active perception and color-coded illumination.
* **"Scorpio-Style" Mobile Manipulator:** High-fidelity simulation in ROS2 Humble featuring a camera-actuated mobile robot arm utilizing active vision.
* **Pneumatic Control Systems:** Comparative analysis of PID, Bang-Bang, and Fuzzy Logic control algorithms for dynamic height stabilization.
